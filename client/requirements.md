## Packages
recharts | Visualization for assessment scores over time
date-fns | Date formatting for session notes
framer-motion | Smooth transitions and animations for a premium feel
react-day-picker | Date picker component (dependency of shadcn calendar)
class-variance-authority | For component variants
clsx | For conditional classes
tailwind-merge | For merging tailwind classes

## Notes
Authentication is handled via Replit Auth (/api/login).
The app uses a sidebar layout for navigation.
SOAP notes have complex nested JSON structures for assessments (PHQ-9, GAD-7).
Speech recognition will use the browser's native API (Web Speech API).
